# Ghost of Babel

**An AI-integrated neural cloud framework**  
Part of Projects ORIGINIUM, RESONANCE, and ECLIPSE.

## Setup

### Install Dependencies

For Python:
```bash
pip install -r requirements.txt
```

For Node.js:
```bash
npm install
```

### Run Locally

Python:
```bash
python app.py
```

Node.js:
```bash
npm start
```
